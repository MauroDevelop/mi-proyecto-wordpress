## Tienda Maro Crochet Bienvenido al repositorio oficial del sitio web de **Maro Crochet**. Este proyecto es una plataforma web desarrollada con WordPress para gestionar y exhibir productos artesanales de tejido a crochet, permitiendo a los usuarios explorar el catálogo y contactar con la marca. ## Acerca del Proyecto Este sitio fue diseñado para consolidar la identidad digital de la marca Maro Crochet. El objetivo principal es ofrecer una interfaz amigable y estética donde se puedan mostrar las creaciones, gestionar contenido de manera dinámica y facilitar la comunicación con los clientes. ### Funcionalidades principales: - **Catálogo de Productos:** Visualización organizada de los tejidos. - **Diseño Responsivo:** Adaptado a móviles, tablets y escritorio. - **Gestión de Contenido:** Administración sencilla a través del panel de WordPress. - **Formulario de Contacto:** Integración para pedidos personalizados. ## Tecnologías Utilizadas - **CMS:** WordPress - **Lenguajes:** PHP, HTML5, CSS3. - **Base de Datos:** MySQL. - **Servidor Local:** Laragon / Apache. - **Control de Versiones:** Git & GitHub. ## Instalación y Despliegue Local Si deseas clonar este proyecto en tu entorno local, sigue estos pasos: 1. **Clonar el repositorio:** git clone [https://github.com/MauroDevelop/mi-proyecto-wordpress.git](https://github.com/MauroDevelop/mi-proyecto-wordpress.git)
